介绍

image-manage是使用springboot基于kubernetes 1.29 (opens new window)的图床项目，简单作为一个尝试

这个项目的初衷是打造一个私人的，简单易操作的图床：

注意

  1. 图床可以单机部署，也可k8s部署
  2. 在使用本项目前，要求你至少简单使用docker

# 特性

  • 伸缩

基于k8s平台，软件实现了简单的手动伸缩的功能

  • WEB管理

拥有web页面，可以轻松管理账号和图片

  • 结构化

借用网盘的样式，为图床提供了简单的浏览和使用

  • PicGo

作文一个图床，适配了知名的picgo插件，在picgo中搜索haowan 即可安装。
